# Binary Classification with a Bank Dataset

Kaggle competition project to predict whether a bank customer
will subscribe to a term deposit.

## Problem Statement
Predict whether a customer will subscribe (`y = 1`) or not (`y = 0`)
based on demographic and marketing campaign data.

## Dataset
- Source: Kaggle Bank Marketing Dataset
- Target variable: `y`

## Approach
- Data preprocessing and categorical encoding
- Feature reduction for computational efficiency
- Probabilistic classification model
- Evaluation using ROC-AUC
- Successful submission to Kaggle leaderboard
